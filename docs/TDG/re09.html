HTML_CSS::createGroup
Prev  Grouping selectors Next

Name

HTML_CSS::createGroup — Creates a new CSS definition group. Returns an integer identifying the group.

Synopsis

      require_once '/CSS.php';
mixed|PEAR_Error HTML_CSS::createGroup( $selectors,  
  $group = NULL);  
string   $selectors;
mixed   $group = NULL;

Description

This package is not documented yet.

Parameter

string $selectors

Selector(s) to be defined, comma delimited.

mixed $group

(optional) Group identifier. If not passed, will return an automatically assigned integer.

See

see HTML_CSS::unsetGroup

Throws

throws HTML_CSS_ERROR_INVALID_INPUT, HTML_CSS_ERROR_INVALID_GROUP

Note

since 0.3.0

This function can not be called statically.
