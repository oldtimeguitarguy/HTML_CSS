Output

HTML_CSS::toArray - Returns the array of CSS properties
HTML_CSS::toInline - Generates and returns the CSS properties of an element or class as a string for inline use.
HTML_CSS::toFile - Generates CSS and stores it in a file.
HTML_CSS::toString - Generates and returns the complete CSS as a string.
HTML_CSS::display - Outputs the stylesheet to the browser.

Send result directly to browser, send it to file or capture to a PHP variable, you will certainly use one of them.
