Searching for selectors and/or properties

With the class constructor, you have ability to set many options :

  • xhtml option defines whether element selectors should be automatically lowercased.
  • tab option defines the string used to indent CSS text. See also: HTML_Common::setTab().
  • filename option allow to parse an external css file.
  • cache option controls caching of the page.
  • online option defines whether to output all properties on one line.
  • groupsfirst option determines whether to output groups before elements.
  • allowduplicates option allow to have duplicate rules in selector.
